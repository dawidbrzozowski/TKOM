Instructions            : Instruction (Instruction)*

Instruction             : Expression SEMICOLON
                        : RETURN Expression?
                        : If-Expression
                        : While-Expression
                        : Function-Definition

If-Expression           : IF LPARENT Expression RPARENT LBRACKET Instructions RBRACKET
                          (ELIF LPARENT Expression RPARENT LBRACKET Instructions RBRACKET)*
                          (ELSE LBRACKET Instructions RBRACKET)*

While-Expression        : WHILE Expression LPARENT Expression RPARENT LBRACKET Instructions RBRACKET

Function-Definition     : FUNCTION IDENTIFIER LPARENT (Function-Argument (COMMA Function-Argument)*)? RPARENT
                        ARROW Return-Type LBRACKET Instructions RBRACKET

Function-Argument       : IDENTIFIER SEMICOLON TYPE

Return-Type             : Type | VOID

Expression              : Assignment
                        : Comparator-Expression ((AND|OR) Comparator-Expression)*

Assignment              : TYPE? IDENTIFIER ASSIGN Expression

Comparator-Expression   : NOT Comparator-Expression
                        : Arithmetic-Expression (Comparator Arithmetic-Expression)*

Arithmetic-Expression   : Term ((PLUS|MINUS) Term)*

Term                    : Factor ((MUL|DIV) Factor)*

Factor                  : (PLUS|MINUS) Factor
                        : Atom

Atom                    : Value|IDENTIFIER
                        : LPARENT Expression RPARENT
                        : Function-Call

Value                   :INT_VALUE|DOUBLE_VALUE|STRING_VALUE|Phys-Value|Unit-Value

Unit-Value              :

Phys-Value              :

Function-Call           : IDENTIFIER LPARENT (Expression (COMMA Expression)*) RPAREN

Type                    : (INT|DOUBLE|STRING|PHYS|UNIT)

Comparator              : (EQ|NOT_EQ|LESS|GREATER|LESS_OR_EQ|GREATER_OR_EQ)
